neovim安裝紀錄

2021-01-22 — Written by Zi-Shane
#neovim 

brew install neovim

安裝neovim

brew install neovim

安裝vim-plugin

sh -c 'curl -fLo "${XDG_DATA_HOME:-$HOME/.local/share}"/nvim/site/autoload/plug.vim --create-dirs \
       https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim'

安裝plugin

esc，進入命令模式（Command mode），用:PlugInstall安裝Plugin

設定檔：https://github.com/Zi-Shane/init.vim

YouCompleteMe報錯

下次打開nvim遇到YouCompleteMe的錯誤，用以下指定解決

python3 -m pip install --user --upgrade pynvim

brew install cmake

python3 ~/.config/nvim/plugged/YouCompleteMe/install.py --clang-completer
